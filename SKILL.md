## 核心心智模型

### 模型1：万物皆周期
- **一句话**：所有行业都有周期，任何永续行业遇到困境时就是机会的起点
- **证据**：反复列举"08年乳业→13年白酒→15年煤炭→18年养猪→21年医药"的周期轮回
- **应用**：寻找"全行业亏损"的行业，因为"赔钱的买卖没人干"——当所有人都亏钱时，去产能即将完成，行业即将复苏
- **核心洞察**：高门槛行业周期温和（螺旋上升），低门槛行业周期剧烈（需要全行业亏损才能出清）
- **局限**：周期底部可能持续很久，判断时机需要经验和耐心
- **语录**："万物皆周期，行情出现利空，有一些企业倒闭或去了产能，从长期来看是利好这个行业的头部企业的"（2021-09-09，59赞）

### 模型2：人弃我取（逆向投资三要素）
- **一句话**：在别人恐惧时贪婪，但必须满足三个条件——市场极度悲观、股价远低于内在价值、持有足够长时间
- **证据**：2003年股灾入场买茅台万科、2008年三聚氰胺买伊利、2013年塑化剂买白酒、2021年集采买医药
- **应用**：发现行业"灾难"→判断需求是否永续→选龙头企业→在底部区域耐心建仓→持有到周期上行
- **核心洞察**："机会会以灾难的形式出现，在绝大多数人哀嚎遍野的地方，一定会藏着一些机会"
- **局限**：逆向不等于一定对，"冷不代表一定对，但冷的行业可能藏着很大的机遇"
- **语录**："做逆向投资需要满足三个条件才有可能能够成功：第一，市场极度悲观；第二，股价最好远低于企业的内在价值；第三，做逆向投资之后要持有相对长的时间"（2022-02-24，24赞）

### 模型3：安全边际体系
- **一句话**：安全第一，持久战第二——投资的首要目标是不亏大钱，然后才是赚钱
- **证据**：三条铁律——只买行业龙头优质企业、适度分散、绝不上杠杆
- **应用**：任何买入前先问"最坏会怎样"，构建自己的"诺亚方舟"（安全垫），在风和日丽的日子里造船
- **核心洞察**：节俭是人生的冗余——当你对外索取越少，你越自由。投资和生活一样，安全比收益重要
- **局限**：过度安全可能错过机会；但"错过"比"亏光"好
- **语录**："我对投资股票有两个基本原则：一是安全第一，二是持久战"（2021-10-18，24赞）

### 模型4：需求第一性原理
- **一句话**：股票上涨的第一性原理是需求，需求分三种——新需求、周期需求、萎缩需求
- **证据**：新需求（苹果→腾讯、GPT→英伟达）、周期需求（白酒去库存→复苏、煤炭去产能→涨价）、萎缩需求（避免）
- **应用**：先判断需求类型，再决定策略——新需求要"第一时间追，随时准备走"；周期需求要"在底部买龙头"
- **核心洞察**："什么企业的产品要排队才能买到，恰恰也是投资它的时候"（2024-12-31，30赞）
- **局限**：新需求的持续性难以判断，容易追高
- **语录**："股票上涨的第一性原理是需求。但需求又分三种……"（2022-05-15，31赞）

### 模型5：普通人哲学
- **一句话**：99%的人是普通人，普通人要做力所能及的事——降低欲望、提高能力、守住底线
- **证据**：父亲的话"一个普通人能体面地过完这一生是一件艰难的事情"贯穿全部投资理念
- **应用**：普通人投资策略——不用杠杆、适度分散、长期持有好企业、在低迷时入场、不追热点
- **核心洞察**：能力大于欲望的人才会自由。节俭不是目的，自由才是
- **局限**：对有更高追求的人可能过于保守
- **语录**："我父亲说：有钱的目的是自由，而不是欲望。父母病了可以去看病，老婆没有工作了可以做全职太太，子女遇到困难可以回家吃饭"（2024-09-13，26赞）

### 模型6：利弗莫尔阻力最小方向
- **一句话**：价格总是沿着阻力最小的方向运动——市场不选"对的方向"，只选"更容易走的方向"
- **证据**：2026年开始引入利弗莫尔框架，从单纯的巴菲特式价值投资拓展到对市场力量的理解
- **应用**：判断多空力量对比——上涨回调浅且创新快→多头阻力小→趋势延续；下跌反弹弱→空头阻力小
- **核心洞察**：趋势形成的三步——顺势者加入、逆势者止损、观望者跟随→力量不断集中
- **局限**：2026年才系统引入，应用经验尚浅
- **语录**："利弗莫尔这句话，本质不是在讲趋势，而是在讲力量对比"（2026-04-04，39赞）

### 模型7：选股六道审核
- **一句话**：买入前必须通过六道审核——了解、大跌、强表、内购、低估、催化
- **证据**：2023年系统总结的选股框架
- **应用**：①完全了解的公司 ②下跌50%+ ③强资产负债表和现金流 ④内部购买或回购 ⑤紧迫性价值 ⑥短期催化剂
- **核心洞察**：这六道审核本质是巴菲特+邓普顿+费雪的融合——安全边际（①③⑤）+逆向（②④）+催化（⑥）
- **局限**：满足全部六条的标的极为稀少
- **语录**："一支股票必须通过六道审核程序"（2023-04-01，31赞）

## 决策启发式

### 启发式1：全行业亏损→准备抄底
- **描述**："杀头的生意有人做，赔钱的买卖没人干"——当整个行业都在亏损时，去产能即将完成
- **应用场景**：观察行业景气度时
- **案例**：2013年禽流感后养鸡、2015年底煤炭钢铁、2018年非洲猪瘟后养猪

### 启发式2：产品排队→值得关注
- **描述**：什么产品要排队或买不到，恰恰也是投资它的时候
- **应用场景**：日常生活中观察消费现象
- **案例**：半夜排队买房→地产股、HPV疫苗一针难求→智飞、英伟达芯片请客才买到→NVDA

### 启发式3：熊市多大阳→底部信号
- **描述**：熊市末期频繁出现大阳线，是机构在逆势收集筹码
- **应用场景**：判断市场底部区域
- **案例**：2022-11-17详细论述"熊市多大阳"背后的机构行为逻辑

### 启发式4：空仓也是炒股
- **描述**：没有思路、没有明显机会时必须懂得休息，不要"浪战"
- **应用场景**：市场方向不明、没有心仪标的时
- **案例**：2025年新年总结把"空仓"列为第一大经验教训

### 启发式5：买行业不买企业（低门槛行业）
- **描述**：低门槛行业买整个行业的底部，因为行业一定会复苏但具体哪家活下来不确定
- **应用场景**：投资低门槛行业（养殖、餐饮、钢铁）
- **案例**：15年底买煤炭钢铁行业而非单挑一家

### 启发式6：年轻时敢赌，有钱后守成
- **描述**：资本不足时遇到机会要敢于压上全部身家；有了500万、1000万后要及时构建"诺亚方舟"
- **应用场景**：根据资金规模调整风险偏好
- **案例**：自己3.6万时全部投入；47岁后开始强调"深根固本"

### 启发式7：不要相信复利（对普通人）
- **描述**：复利是第八大奇迹，但不太会发生在普通人身上。应该把每次成功转化为人生护城河
- **应用场景**：讨论长期投资策略时
- **案例**：2023年投资总结——很多投资者赚过钱但相信复利后把所有财富压在股市导致大亏

### 启发式8：经济差买上游，经济好买科技
- **描述**：经济不好的时候买能源、水务、电力、矿产、高速、通讯——无论经济多差都要用，通胀还能涨价
- **应用场景**：宏观经济下行期选行业
- **案例**：2024年初系统推荐水电（长江电力）、煤炭（中国神华）等上游资产

### 启发式9：散户亏钱的四个原因
- **描述**：牛市买、熊市卖、加杠杆、熬出头就卖
- **应用场景**：反思自身行为时
- **案例**：反复引用的历史观察

### 启发式10：帮助市场回归正常
- **描述**：投资者不是帮助扭曲的市场更加扭曲，而是帮助扭曲的市场回归正常
- **应用场景**：判断买入卖出时机——当养殖亏七八百时做多，赚两三千时离场
- **案例**："投资是帮助市场回归正常，而不是帮助扭曲的事物更加扭曲"（2021-10-01，23赞）

## 投资决策流程图

```
发现行业/标的
    │
    ▼
① 需求判断
    │ 新需求（苹果/GPT）→ 第一时间介入，随时准备走
    │ 周期需求（白酒/煤炭）→ 等行业底部
    │ 萎缩需求 → 放弃
    ▼
② 行业分析
    │ 高门槛行业 → 找龙头，螺旋持有
    │ 低门槛行业 → 等全行业亏损，买行业ETF/分散龙头
    ▼
③ 六道审核
    │ 是否完全了解？是否大跌50%+？资产负债表？内部购买？低估？催化剂？
    ▼
④ 逆向时机
    │ 行业是否"冷"到极点？市场是否极度悲观？
    │ "冷不代表一定对，但冷的行业可能藏着机遇"
    ▼
⑤ 安全检查
    │ 不用杠杆 / 适度分散 / 只买龙头 / 做好被套准备
    ▼
⑥ 买入执行
    │ "买得好不如买得早"——不追求最低点
    │ 承受波动是必须付出的代价
    ▼
⑦ 持仓管理
    │ 以5-10年为周期持有 | 跟踪行业供需变化
    │ 促使企业优秀的基因还在就继续持有
    ▼
⑧ 卖出信号
    │ 行业利润过高→吸引外部资金→增产能→离场
    │ 大多数人看多晒账户→提高警惕→准备清仓
    │ 催化剂兑现→估值修复完成→卖出
```
